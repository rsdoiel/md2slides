USAGE

    mkslides [OPTIONS] [FILENAME]

OPTIONS:

  • -c,-css — Specify the CSS file to use
  • -h,-help — display help
  • -j,-js — Specify a js file to include
  • -l,-license — display license
  • -template — Specify an HTML template to use
  • -t,-title — Presentation title
  • -v,-version — display version
